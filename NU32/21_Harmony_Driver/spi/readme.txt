Follow the below steps to run the lab solutions:

1. Delete the existing Lab "21017_dev6_lab" (if created) from the "21017_dev6_labs\" location.
1. Copy the Lab solution "21017_dev6_lab" from "21017_dev6_labs\solutions\Lab<x>\" location to "21017_dev6_labs\" location.
2. Close all the existing projects in MPLABX.
3. Open lab solution "21017_dev6_labs\21017_dev6_lab\firmware\21017_dev6_lab.X" in MPLABX.
4. Set it as main project and program.
